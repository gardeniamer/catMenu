# 猫谱APP：

#### 	1.特点：

​			●利用uni-app (基于vue2) 开发完成的一个前端化应用。

​			●猫咪的面部建档与读档利用了**悦保科技文档**([文档中心 - 悦保科技文档中心 (ybinsure.com)](http://ybinsure.com/spa/documentCenter/ocr-finance/mark))里的公开API。

#### 	2.功能：

​			●猫咪面部建档与读档  (相机与相册功能均可)。

​			●日历的打卡与标记。

​			●聊天框的内容输出。

​			●内容的延迟展示。

​			●页面跳转内容的多样化展示。

​			●用户的登录。
